Suggested coding picks

Tip: model names come from WebLLM’s registry. If you need a model not present there, add a custom model record after compiling it with MLC.
Offline behavior: the model and runtime assets can be cached after the first successful load. For fully self-hosted offline use, package the JS module and model assets locally.
